K-Razy Shoot-Out Font Usage Guide
========================================

This font contains the original 8x8 pixel characters from
the 1981 Atari 5200 game K-Razy Shoot-Out.

Available Characters:
  Punctuation: *+,-./:;?
  Numbers: 0123456789
  Letters: ABCDEFGHIJKLMNOPQRSTUVWXYZ
  Space: (available)

Sample Text:
  KRAZY SHOOT-OUT
  SCORE: 12,345
  TIME: 05:30
  GAME OVER.
  HIGH SCORE + BONUS
  LEVEL 1/10
  PLAYER * 3
  READY; SET; GO
  CONTINUE?

Files Created:
  - krazy_font_bitmap.png (complete font bitmap)
  - character_samples/ (individual character images)
  - krazy_character_map.txt (ASCII art map)
  - krazy_usage_guide.txt (this file)

Usage Ideas:
  - Use character images in game development
  - Create retro-style graphics and logos
  - Reference for pixel art and font design
  - Educational study of classic game graphics

Technical Details:
  - Original size: 8x8 pixels per character
  - Format: 1-bit monochrome bitmap
  - Total characters: 89 (46 mapped to ASCII)
  - Source: K-Razy Shoot-Out ROM data
  - Includes: letters, numbers, punctuation (* + , - . / : ; ?)
