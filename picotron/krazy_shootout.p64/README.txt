K-RAZY SHOOT-OUT
Picotron Edition

A faithful recreation of the Atari 5200 classic.

CONTROLS:
- Arrow Keys: Move (8 directions)
- Z or X: Fire weapon (hold + direction to aim)
- ESCAPE: Pause/Exit

GAME FILES:
- main.lua: Entry point and game loop
- utils.lua: Helper functions
- sprites.lua: Sprite rendering
- collision.lua: Collision detection
- arena.lua: Arena generation
- entities.lua: Player, enemies, missiles
- hud.lua: HUD and timer display
- gfx/0.gfx: Sprite data (placeholder)
- sfx/0.sfx: Sound effects (placeholder)

RESOLUTION: 320x192 (matching Atari 5200 Mode 8)

Based on disassembly of the original Atari 5200 game.
Original game by K. Dreyer (CBS Software, 1981)
